## About
This is a dashboard made using Next.js + Tailwind CSS + MongoDB.
The website includes 6 pages, each with a different task.
> - Homepage - to show us data about various revenue, total number of users and products and active users. Also a table with recent orders and a bar chart with user salaries.
> - Users page - to show us a table with all user information and update and delete function
> - Products page - to show us all the products with their data and a pie chart with number of products. Also search and sort function
> - Orders page - to show us all user orders with search order and detail display feature on mouseover
> - Add page - allows us to add a new order (validation form)
> - Update page - Allows us to update user data

### The project includes:
> - CRUD operations
> - validation form
> - search and sort functionality
> - real database data
> - modern and responsive design

## How to run
This is a [Next.js](https://nextjs.org/) project bootstrapped with `create-next-app`, before cloning this repo please ask !
* Clone the repo
```bash
git clone
```
* Install dependencies
```bash
npm i
```
* Create an .env file
```bash
Create an .env file with your own API key
```
